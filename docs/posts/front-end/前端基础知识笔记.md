## html

### 图片格式
|格式|说明|
|-|-|
|.bmp|Windows画图软件默认保存的格式,位图|
|.gif|支持动画|
|.jpeg|有损压缩图片,用于照片|
|.png|便携式网络图片,用于logo,背景图片,支持透明和半透明|
|.svg|矢量图|
|.webp|最新的压缩算法非常优秀的图片格式|
