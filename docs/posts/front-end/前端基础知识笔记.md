### 浮动与定位
`本质功能是实现并排`
- 子盒子会按照顺序进行贴靠,如果没有足够空间，则会寻找再前一个兄弟元素
- 浮动元素不再区分块级元素行内元素，已脱离文档流，一律能设置宽高
- 垂直显示的盒子不要设置浮动，只有并排显示的盒子才要设置浮动
- 左右并排浮动时,父元素要清除浮动 
```css
/* 父 */
overflow: hidden;
/* 子 */
float:left;
/* 子 */
float:right;
```

### 函数提升
::: warning

- 这是函数提升

- 函数比变量优先提升
- arguments参数类数组
:::

```js

fun()

var fun = function(){alert(1)}
```

### 闭包
- 一个函数返回另一个函数
- 相当于把函数内部的函数,转移到外部执行
- 闭包是函数本身和该函数声明时所处的环境状态的组合
- 函数+所处环境的变量(能够记忆住其定义时候所处的环境)
- 每次创建函数时都会创建闭包
- 但是闭包特性往往需要将函数换一个地方执行,才能被观察出来
```js
function fun(){
  var name = "慕课网"

  function innerFun(){
    console.log(name)
  }
  return innerFun
}

var inn = fun();
inn();
```

#### 闭包的用途
- 记忆性:当闭包产生时,函数所处环境的状态会始终保持在内存中，不会在外层函数调用后被自动清除.
```js
function createCheckTemp(standardTemp){
  function checkTemp(n){
    if(n > standardTemp){
      console.log('偏高')
    }else{
      console.log('正常')
    }
  }
  return checkTemp
}

var a = createCheckTemp(37)
var b = createCheckTemp(36)
a(36.5); // 正常
b(36.5); // 偏高
```
- 模拟私有变量,类似Java中对象的私有成员
```js
// 让变量私有化
function fun(){
  var a = 0;
  return {
    getA: function(){
      return a;
    }
  }
}

var obj = fun()
console.log(obj.getA())
```

#### 作用2: 将全局变量变成局部变量
不使用IIFE
```js

  arr.push(function(){

  })
```

使用IIFE

```js
      })
    }
  )(i)
}

arr[2]() //2
arr[3]() //3
```
