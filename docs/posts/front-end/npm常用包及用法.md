## [nodemon](https://www.npmjs.com/package/nodemon)

### 安装
```bash
npm install -g nodemon
```

### 简单用法
```json title="package.json"
"scripts": {
  "test": "echo \"Error: no test specified\" && exit 1",
  "start": "nodemon --watch main.js --exec \"electron .\""
},
```
