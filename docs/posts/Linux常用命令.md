# Linux/Mac常用命令

## 文件操作
```bash
mv a b
cp a b
mkdir dir_name
touch file_name
```

## 一些简单的命令
```bash
# 输出环境变量
echo $PATH

# 输出home目录
echo $HOME

# 存储环境变量(只能在terminal未关闭的情况下,临时存储)
export PASSWORD=password

# 永久存储环境变量(~/.zshrc添加)
export PASSWORD=password
```
