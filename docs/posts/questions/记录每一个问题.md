## 技术类
### 为什么a:link没有生效
```html
 <a href="#">我是伪类</a>
```
```css
  a:link{
    color: red;
  }

  a:visited{
    color: rgb(0, 255, 153);
  }

  a:hover{
    color: yellow;
  }

  a:active{
    color:pink;
  }
  ```
答:`因为a[href]没有写真正的链接地址,如改成href="www.baidu.com"即可`

