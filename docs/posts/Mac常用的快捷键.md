# Mac使用小技巧
::: warning
以下快捷键在Mac下使用,并且在系统设置里把
- Command键映射为Ctrl键
- Ctrl键映射为Command键
本人键盘使用的XS75T
Alt键是option键
:::


## 快捷键
::: warning
如果没有向我上面把修饰更改,那么你需要把下面的
Command 与 Ctrl对换才能生效
:::
| 快捷键 | 功能 | 功能键或自定义或软件 |
| ------ | ------ | ------ |
| Ctrl + ↑ | 光标移动到文档的开头 | Home |
| Ctrl + ↓ | 光标移动到文档的结尾 | END |
| Ctrl + ← | 光标移动到文档的结尾 | 行首 |
| Ctrl + → | 光标移动到文档的结尾 | 行尾 |
| Command + ↑ | 缩窗口到窗口列表 | 无 |
| Command + ↓ | 从窗口列表放大 | 无 |
| Command + ← | 向左选择窗口列表 | 无 |
| Command + → | 向→选择窗口列表 | 无 |
| Ctrl + , | 切换输入法 | 自定义 |
| Ctrl + Shift + . | 显示或隐藏文件 | 自定义 |
| Command + Space | 快速搜索软件或问题 | Alfred |
| Ctrl + Shift + v | 剪切板历史 | paste软件 |
